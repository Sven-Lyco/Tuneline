# Tuneline

Digitales Musik-Timeline-Partyspiel (wie Hitster).

- React 19 + TypeScript, Emotion CSS-in-JS, Vite
- Spotify PKCE OAuth fur Login + Playlist-Auswahl durch den Host
- Song-Metadaten (Titel, Interpret, Jahr, Cover) aus der Spotify Web API
- Audio aktuell: Spotify Web Playback SDK (ab 60s, erfordert Premium)
- Audio geplant: iTunes Search API (30s Previews, kein Auth, multiplayer-fahig)
- 2-6 Spieler, konfigurierbare Rundenzahl, Punktesystem
- Geplant: Online-Multiplayer via WebSockets / Socket.io mit Einladungslinks
- Design: Dunkles Neon-Theme, Space Mono + Outfit Fonts
